# Salsa

* __개발 내용__ : 인프라를 이해하고 있는 사람만 수동으로 셋팅 가능한 것을, 누구든 직관적으로 과카몰리 서버의 전체적인 세팅을 쉽게 할수 있는 툴 제작

* __개발 목적__ : 과카몰리의 세팅을 운영자의 요구대로 빠르게, 실수 없이 세팅 할 수 있도록 함.
